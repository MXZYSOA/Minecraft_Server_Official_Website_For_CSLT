组件使用演示

HTML 结构

在页面中使用组件非常简单，只需要添加带有 data-component 属性的元素：

<!-- 导航栏组件 -->
<div data-component="navbar"></div>

<!-- 页面内容 -->
<main>
    <!-- 你的内容 -->
</main>

<!-- 页脚组件 -->
<div data-component="footer"></div>

<!-- 组件加载器 -->
<script src="components/component-loader.js"></script>

自动加载

组件会在页面加载完成后自动加载，无需额外的JavaScript代码。

  • 导航栏会自动高亮当前页面
  • 页脚会自动更新在线人数
  • 所有交互功能都会正常工作

动态控制

你也可以通过JavaScript手动控制组件的加载：

// 手动加载所有组件
loadComponents().then(() => {
    console.log('组件加载完成');
});

// 重新加载特定组件
reloadComponent('navbar');

优势

  • 统一性：所有页面使用相同的组件
  • 维护性：只需修改一个文件
  • 复用性：可在任何页面中使用
  • 灵活性：支持动态加载和重载
  • 简单性：使用简单，无需复杂配置
  • 扩展性：易于添加新组件
